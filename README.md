## General

This project is a dashboard to manage player registrations for a game called "Arma 3", for the "7th Company" community. This part is the back-end of the project.

## Technical

I used NodeJS with ExpressJS and MongoDB.

## Install

```sh
git clone https://github.com/7eCompagnie/arma-website-api.git
```

## Usage

```sh
yarn install
node src/index.js
```
